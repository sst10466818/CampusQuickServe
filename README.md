# Campus Quick Serve

## project overview

This project is a small Java prototype application that demonstrates basic Object-Oriented Programming (OOP) principles using a simple graphical user interface.

The application allows users to enter customer details, place an order, calculate the order total, and display a formatted receipt. Instead of using a console-based interface, the program uses JOptionPane dialogs to create a GUI-based interaction.

The system is structured using multiple classes and methods to ensure clean code organization and to demonstrate proper OOP design rather than placing all logic inside the main method.
